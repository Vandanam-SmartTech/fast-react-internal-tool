## Backend Processing Flow

```
┌─────────────────────────────────────────────────────────────┐
│              GET /api/bootstrap?page=0&size=9               │
│              Authorization: Bearer <JWT>                     │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │    Extract JWT Claims (No DB Query)   │
        │    - username                         │
        │    - global_roles                     │
        │    - org_roles                        │
        └───────────────────────────────────────┘
                            │
                ┌───────────┴───────────┐
                │                       │
                ▼                       ▼
    ┌──────────────────┐    ┌──────────────────┐
    │  SUPER_ADMIN?    │    │  ORG_ADMIN?      │
    └──────────────────┘    └──────────────────┘
                │                       │
                ▼                       ▼
    ┌──────────────────┐    ┌──────────────────┐
    │ Get ALL users    │    │ Get paginated    │
    │ Get ALL orgs     │    │ users for org    │
    │ Extract roles    │    │ Extract roles    │
    └──────────────────┘    └──────────────────┘
                │                       │
                └───────────┬───────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │         Cache Response (30min)        │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │         Return JSON Response          │
        └───────────────────────────────────────┘
```

## Response Structure by Role

### SUPER_ADMIN Response
```json
{
  "users": [
    { "id": 1, "username": "admin", ... },
    { "id": 2, "username": "user2", ... }
  ],
  "organizations": [
    { "id": 1, "name": "Org1", ... },
    { "id": 2, "name": "Org2", ... }
  ],
  "roles": [
    { "name": "ROLE_SUPER_ADMIN" },
    { "name": "ROLE_ORG_ADMIN" }
  ]
}
```

### ORG_ADMIN Response
```json
{
  "usersPaginated": {
    "content": [
      { "id": 5, "username": "orguser1", ... },
      { "id": 6, "username": "orguser2", ... }
    ],
    "totalElements": 25,
    "totalPages": 3,
    "number": 0,
    "size": 9
  },
  "totalUsers": 25,
  "roles": [
    { "name": "ROLE_ORG_ADMIN" },
    { "name": "ROLE_STAFF" }
  ]
}
```

## Caching Strategy

```
┌─────────────────────────────────────────────────────────────┐
│                    First Request                             │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │    Cache Key: token_page_size         │
        │    Example: "Bearer_xyz_0_9"          │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │         Query Database                │
        │         Process Data                  │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │    Store in Cache (30min TTL)         │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │         Return Response               │
        └───────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                  Subsequent Requests                         │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │    Check Cache (same key)             │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │    Return Cached Response (instant)   │
        │    No Database Query                  │
        └───────────────────────────────────────┘
```

                            │

        ┌───────────────────────────────────────┐

                            ▼

                            ▼
