**New to the project?**
1. Start with [QUICKSTART.md](./QUICKSTART.md)

3. Run `npm run dev`

**Want to understand optimizations?**
1. Read [SUMMARY.md](./SUMMARY.md) for overview

**Deploying to production?**
1. Follow [CHECKLIST.md](./CHECKLIST.md)
2. Reference [COMMANDS.md](./COMMANDS.md)

**Migrating existing code?**

## 📝 Quick Reference

### Commands
```bash
npm run dev              # Development
npm run build            # Production build
npm run build:analyze    # Analyze bundle
npm run preview          # Test production
```

### Key Metrics
- Initial Bundle: ~500KB
- Performance Score: 90+
- FCP: <1.5s
- LCP: <2.5s
- TTI: <3s

### Success Criteria
✅ Performance: 90+
✅ Bundle: <500KB
✅ Chunks: 15-20
✅ Load Time: <3s
